🏧 ATM Management System (Python)

A simple ATM Management System built using Python with Object-Oriented Programming (OOP) concepts. This project simulates the basic functionality of a real ATM while demonstrating the use of Encapsulation, loops, methods, and exception handling.

🚀 Features
PIN Authentication
Check Account Balance
Deposit Money
Withdraw Money
Change ATM PIN
Transaction History
Menu-Driven Interface
Encapsulation using Private Variables
Input Validation
Multiple Login Attempts
🛠️ Technologies Used
Python 3
Object-Oriented Programming (OOP)
Encapsulation

▶️ How to Run
Clone this repository.
git clone https://github.com/YOUR_USERNAME/atm-management-system.git
